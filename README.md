# Task 5 - Network Packet Analyzer

## Objective
Develop a packet sniffer tool that captures and analyzes network packets.

## Features
- Captures network packets using Scapy.
- Displays source IP addresses.
- Displays destination IP addresses.
- Shows protocol information.
- Displays payload details.
- Captures 5 packets for educational purposes.

## Technologies Used
- Python
- Scapy
- Npcap

## Educational Purpose
This project was developed for learning network packet analysis and should be used ethically.
